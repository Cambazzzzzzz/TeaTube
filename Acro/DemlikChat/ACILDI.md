# 🎉 DemlikChat Başarıyla Açıldı!

## ✅ Sunucu Çalışıyor

Sunucu **port 3001**'de çalışıyor.

## 🌐 Tarayıcıdan Aç

Tarayıcını aç ve şu adrese git:

```
http://localhost:3001
```

## 📱 Kullanım

### 1. Kayıt Ol
- "Kayıt Ol" sekmesine tıkla
- Kullanıcı adı, görünen ad ve şifre gir
- Kayıt ol butonuna tıkla

### 2. Giriş Yap
- "Giriş Yap" sekmesine geç
- Kullanıcı adı ve şifre ile giriş yap

### 3. Arkadaş Ekle
- Sol panelde "Arkadaş Ekle" butonuna tıkla
- Arkadaşının kullanıcı adını gir

### 4. Mesajlaş
- Arkadaşını seç
- Mesaj yaz ve Enter'a bas
- Fotoğraf göndermek için + butonuna tıkla

## 🎨 Tema Değiştir

Sağ alt → Ayarlar → Tema bölümünden 9 farklı tema arasından seçim yap!

## 🧪 Test İçin İki Kullanıcı

İki kullanıcı ile test etmek için:
1. Normal pencerede bir kullanıcı oluştur
2. Gizli pencere (Ctrl+Shift+N) aç
3. İkinci kullanıcı oluştur
4. Birbirlerine arkadaşlık isteği gönder
5. Mesajlaş!

## 🛑 Sunucuyu Durdurmak İçin

Terminal'de `Ctrl + C` tuşlarına bas.

## 🔄 Sunucuyu Yeniden Başlatmak İçin

```bash
cd DemlikChat
node server.js
```

## 📊 Sunucu Bilgileri

- Port: 3001
