# 🚀 DemlikChat - Hızlı Başlangıç

## 📦 Kurulum (3 Adım)

### 1️⃣ Bağımlılıkları Yükle
```bash
cd DemlikChat
npm install
```

### 2️⃣ Sunucuyu Başlat
Yeni bir terminal aç:
```bash
npm run dev
```

### 3️⃣ Uygulamayı Aç
Başka bir terminal aç:
```bash
npm start
```

## 🎯 İlk Kullanım

1. **Kayıt Ol** sekmesine tıkla
2. Bilgilerini gir (kullanıcı adı, görünen ad, şifre)
3. **Kayıt Ol** butonuna tıkla
4. **Giriş Yap** sekmesine geç
5. Giriş yap ve keyfini çıkar! 🎉

## 💬 Hızlı Kullanım

- **Arkadaş Ekle**: Sol panel → "Arkadaş Ekle" → Kullanıcı adı gir
- **Mesaj Gönder**: Arkadaş seç → Mesaj yaz → Enter
- **Fotoğraf Gönder**: + butonuna tıkla → Fotoğraf seç
- **Grup Oluştur**: "Gruplar" sekmesi → "Grup Oluştur"
- **Ayarlar**: Sağ alt → "Ayarlar" butonu

## 🎨 Tema Değiştir

Ayarlar → Tema bölümü → İstediğin temayı seç

## 📱 Mobil Kullanım

Tarayıcıdan `http://localhost:3000` adresine git
