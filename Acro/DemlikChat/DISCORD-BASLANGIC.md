1. **simple-peer eklendi** - Sesli özellik için

## 🚀 Şimdi Yapılacaklar

### 1. Discord UI (2-3 saat)
- Sol sidebar: Sunucu listesi (DC logosu)
- Orta sidebar: Kanal listesi  
- Ana alan: Mesaj alanı
- Sağ sidebar: Üye listesi
- Discord renkleri (#2C2F33, #23272A, #7289DA)

### 2. Sunucu Sistemi (1 saat)
- Sunucu oluşturma
- Sunucu ayarları
- Sunucu davetleri

### 3. Kanal Sistemi (1 saat)
- Metin kanalları
- Ses kanalları
- Kanal kategorileri

### 4. Sesli Özellik (3-4 saat)
- WebRTC kurulumu
- Mikrofon erişimi
- Ses akışı
- Kontroller (mute, deafen)
- Konuşan göstergesi

## 📦 Kurulum

```bash
cd DemlikChat
npm install
npm start
```
