# 🚀 DEMLIKCHAT - QUICK START GUIDE

## ⚡ 5-Minute Setup

### 1️⃣ Start the Server
```bash
cd TeaTube
node server.js
```

### 2️⃣ Open DemlikChat
Open your browser and go to:
```
http://localhost:3456/demlikchat/app
```

### 3️⃣ Create Account
- Click "Kayıt Ol" (Register)
- Enter username and password
- Click "Kayıt Ol" button

### 4️⃣ Create Server
- Click the "+" button in server list
- Enter server name
- Click "Oluştur" (Create)

### 5️⃣ Start Chatting!
- Type a message in the text channel
- Press Enter to send
- See messages appear in real-time!

## 🎤 Voice Chat Quick Start

### Join Voice Channel
1. Click on a voice channel (🔊 icon)
2. Allow microphone access when prompted
3. You're now in voice chat!

### Voice Controls
- **Mute/Unmute:** Click microphone icon
- **Deafen/Undeafen:** Click headphone icon
- **Leave:** Click "Ayrıl" button

## 💬 Key Features

### Messaging
- **Send Message:** Type and press Enter
- **Edit Message:** Right-click → Edit
- **Delete Message:** Right-click → Delete
- **Add Reaction:** Click 😊 icon on message
- **Pin Message:** Right-click → Pin

### Friends
- **Add Friend:** Click user → Add Friend
- **Accept Request:** Click ✓ in friend requests
- **Send DM:** Click friend → Send Message

### Roles (Admin Only)
- **Create Role:** Server menu → Roles → Create
- **Assign Role:** Right-click member → Add Role
- **Edit Permissions:** Click role → Edit

### Search
- **Search Messages:** Click 🔍 icon
- **Type query:** Enter search term
- **View Results:** Click result to jump to message

## 🔧 Troubleshooting

### Voice Not Working?
1. ✅ Check microphone permissions
2. ✅ Use Chrome/Edge (best support)
3. ✅ Check if microphone is working in other apps
4. ✅ Refresh the page

### Effects Not Working?
1. ✅ Join voice channel first
2. ✅ Update your browser
3. ✅ Try different effect
4. ✅ Check CPU usage (close other apps)

### Can't Connect?
1. ✅ Check if server is running (`node server.js`)
2. ✅ Check port 3456 is not in use
3. ✅ Try `http://localhost:3456/demlikchat/app`
4. ✅ Clear browser cache

### Messages Not Sending?
1. ✅ Check internet connection
2. ✅ Refresh the page
3. ✅ Check browser console for errors
4. ✅ Restart server
